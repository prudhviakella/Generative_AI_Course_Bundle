# ------------------------------------------------------------
# 1. Navigate to your project root directory:
#       cd <project-root-folder>
# 2. Enter the virtual environment folder:
#       cd .venv
# 3. Go into the 'bin' directory:
#       cd bin
# 4. Activate the virtual environment:
#       macOS/Linux:  source activate
